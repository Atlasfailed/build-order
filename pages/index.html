Important: How to Interpret This Data

Build Commands vs. Actual Builds
This data shows build commands (queues), not actual construction starts or completions. Commands that were queued may never have been executed.
Example: A player queuing all mexes on the map would show up even if they only built a few.
No Game Settings Filter
Build orders from all game settings are included: evolved commanders, ruins, lava maps, water maps, etc.
Example: People starting out with tidals.
Low Sample Size Builds Included
Builds with very few players are still shown. Interpret these with caution as they may be caused by weird game settings.
These builds are still included because they might also be unique strategies by high skill players.
